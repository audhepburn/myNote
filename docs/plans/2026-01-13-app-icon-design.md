# myNote App Icon Design

**Date:** 2026-01-13
**App:** myNote - 个人时间线笔记

---

## 视觉概念

**核心元素：** 简化的便签纸与时间线的结合

一个圆角矩形（便签纸）被一条柔和的对角线分割，左上部分是浅橙色，右下部分是深橙色。对角线暗示"时间流逝"的概念，便签形状代表"笔记"。整体保持极简，无多余装饰。

---

## 详细规格

### 尺寸
- **基础尺寸:** 1024×1024px
- **圆角半径:** 180px
- **安全区域:** 内边距约 80px

### 色彩
```swift
// 浅橙色
let lightOrange = Color(red: 1.0, green: 0.62, blue: 0.35)
// 深橙色
let darkOrange = Color(red: 1.0, green: 0.42, blue: 0.16)
// 渐变角度: 45度 (从左上到右下)
```

### 图形构成

1. **背景**
   - 1024×1024 圆角矩形
   - 45度线性渐变填充

2. **分割线** (时间线隐喻)
   - 从左上(0,0)到右下(1024,1024)
   - 白色，30%不透明度
   - 宽度: 3px

3. **内容线条** (笔记隐喻)
   - 3条水平白色线条
   - 第一条: y=440, 长度300px, 高度8px
   - 第二条: y=480, 长度240px, 高度8px
   - 第三条: y=520, 长度180px, 高度8px
   - 圆角: 4px
   - 不透明度: 60%
