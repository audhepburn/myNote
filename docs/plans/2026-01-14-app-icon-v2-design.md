# myNote App Icon Design v2

**Date:** 2026-01-14
**App:** myNote - 个人时间线笔记

---

## 视觉概念

**核心元素：** 拟人化的笔记本时钟

一个圆润的橙色笔记本，有着可爱的卡通笑脸和时钟指针，采用温暖的手绘风格。笔记本既是记录的载体，又是时钟，完美诠释"时间线笔记"的概念。

---

## 详细规格

### 风格
- **手绘插画风格**
- **边缘不规则**：±4px 自然抖动
- **线条粗细变化**：8-12px 范围
- **温暖可爱**

### 尺寸
- **基础尺寸:** 1024×1024px
- **主体圆形:** 中心 (512, 512), 直径 780px
- **安全区域:** 外边距 122px

### 色彩
```swift
// 主色调
笔记本主体: #FF9F5A (浅橙) → #FF8A40 (深橙渐变)
眼睛: #1E3A5F (深蓝)
微笑/指针: #F4D03F (金黄)
背景: #FFF3E0 (淡奶黄)
腮红: #FFB6C1 (粉桃)
```

### 元素布局

**1. 笔记本主体**
- 圆形，直径 780px
- 橙色渐变（从中心向外）
- 手绘不规则边缘
- 纸张纹理噪点

**2. 面部特征**
- **左眼:** (430, 420), 直径 24px, 略小
- **右眼:** (594, 420), 直径 26px, 略大
  - 白色眼仁 + 深蓝瞳孔
  - 高光点增加立体感
- **微笑:** 从 (400, 580) 经 (512, 610) 到 (624, 570)
  - 金黄色，10px 粗细
  - 自然曲线，略不规则
- **腮红:** 淡粉色随机点缀

**3. 时钟指针**
- **时针:** 指向 2 点方向，粗 8px，金黄色，末端 14px 圆点
- **分针:** 指向 8 分钟位置，粗 6px，金黄色，末端 12px 圆点
- **起点:** 从中心 (512, 512) 发出

### 手绘效果
- **边缘抖动:** ±4px 模仿手绘不完美
- **曲线不规则:** 贝塞尔曲线带有随机偏移
- **纹理叠加:** 5000 个随机噪点增加纸张质感

---

## 设计理念

**为什么选择笔记本时钟 + 笑脸？**
- 📝 **笔记本** = 笔记应用的核心功能
- ⏰ **时钟指针** = 时间线概念
- 😊 **笑脸** = 友好、亲切、记录美好瞬间
- 🎨 **手绘** = 温暖、人情味、不同于冷冰冰的科技感

---

## 与原设计对比

| 特征 | 原设计 | 新设计 |
|------|--------|--------|
| 风格 | 极简抽象 | 手绘插画 |
| 元素 | 对角线+文本 | 拟人化角色 |
| 情感 | 冷静专业 | 温暖亲切 |
| 识别度 | 需要解释 | 一目了然 |

---

## 技术实现

- 生成工具: Python PIL
- 手绘不规则度: 随机偏移算法
- 渐变模拟: 多层圆叠加
- 纹理: 随机噪点叠加
